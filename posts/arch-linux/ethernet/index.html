Ethernet

November 5, 2018

Ethernet does not necessarily work out of the box - in which case the lights on the ethernet port will not even blink. To enable: See ethernet devices

$ ip link

Then enable the ethernet device

$ ip link set dev <devicename> up

is the one that looks something like “enp0s31f6”

Then get DHCP working for that interface. I am able to just run

% sudo dhcpcd
