Copying and Pasting in Arch linux with Suckless Simple Terminal

Patch Simple Terminal so that there is just a single clipboard. https://st.suckless.org/patches/clipboard/ If like me, you had never patched a C file before… cd src/st-0.8.1 wget https://st.suckless.org/patches/clipboard/st-clipboard-0.8.1.diff patch < st-clipboard-0.8.1.diff Amazing! If like me, maybe you also weren’t aware that copy and paste inside ST uses the keys: Control-Shift C (copy) and Control-Shift V (paste) Install Chris Down’s clipmenu (AUR) clipnotify (AUR) is a dependency of clipmenu. With AUR, dependencies that are also AUR / community sourced have to be installed manually.

Ethernet

Ethernet does not necessarily work out of the box - in which case the lights on the ethernet port will not even blink. To enable: See ethernet devices $ ip link Then enable the ethernet device $ ip link set dev <devicename> up is the one that looks something like “enp0s31f6” Then get DHCP working for that interface. Fonts in Arch

It’s pretty easy to set font in the Virtual Console. pacman -S terminus-font setfont ter-v32n Voila, eyes saved. I found it a bit trickier setting the font within X / dwm / st. On the T460S I used X with dwm as the window manager, and st (simple terminal) to get the all important tty. Initially, the fonts inside my ttys were screwy - as in microscopic and also the kerning was totally wack.

Installing Dropbox w Cryptomator

https://wiki.archlinux.org/index.php/Dropbox pacman -S dropbox I just run Dropbox when I’m in Xorg, so using ~/.xinitrc: /usr/bin/dropbox & The next time you start X, Dropbox will pop open a browser to sign in with. After you sign in, ~/Dropbox will just automatically sync. Cryptomator lets you store encrypted vaults on Dropbox. Unlocked vaults are made available via webdav, so some type of webdav client is needed. I tried nautilus and it worked, so I am using that.
